# 📘 Assignment: CSV File Processing with Python

## 🎯 Objective

Practice reading, filtering, and writing CSV data using Python to build a small data-processing workflow.

## 📝 Tasks

### 🛠️ Load and Explore CSV Data

#### Description
Read a CSV file and summarize its contents to understand the data structure and values.

#### Requirements
Completed program should:

- Open and read a CSV file using Python's `csv` module
- Display the headers and the number of rows in the dataset
- Print the first 5 rows of data
- Calculate and display a simple summary, such as the count of records or a numeric total


### 🛠️ Filter and Transform Rows

#### Description
Filter the dataset based on a condition and transform selected fields for further analysis.

#### Requirements
Completed program should:

- Filter rows according to a specific condition (for example, values above a threshold or a matching category)
- Create a transformed output for each selected row, such as converting string values or calculating a new field
- Store the filtered and transformed rows in memory for writing later


### 🛠️ Write Processed Data to a New CSV

#### Description
Save the filtered and transformed results to a new CSV file and report the final output.

#### Requirements
Completed program should:

- Write the processed rows to a new CSV file with appropriate headers
- Preserve the output format so it can be opened in a spreadsheet
- Print a summary message with the number of rows written and the output filename
- Include example usage in the README if helpful
